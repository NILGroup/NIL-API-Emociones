Palabra - Emoción mayoritaria

Devuelve la emoción con mayor grado, junto con el grado, dada una palabra. Si la palabra no existe en el diccionario afectivo, lo indica.

La URL de la petición es: http://sesat.fdi.ucm.es/emociones/palabra/mayorEmo/

Se obtiene la respuesta en JSON

Nombre Requerido Descripcion Valor por defecto
NI (No importa) Sí Es la palabra que queremos buscar No tiene valor por defecto

http://sesat.fdi.ucm.es/emociones/palabra/mayorEmo/abejas/

En caso de ser una palabra que aparezca en el diccionario:

{ "grado": 3.47, "emociones": "Miedo" }

En caso de no aparecer en el diccionario

{ "detail": "Not found." }
